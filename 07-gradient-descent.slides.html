  • $w_1$ $\frac{\mathrm \partial}{\mathrm \partial w_1} J(w_1)$ $w_1$

第三種找出 $w$ 的方法：梯度遞減（Gradient Descent）指的就是從隨機指定的 $w_1$ 起始透過迭代、有意識地朝 $w_1^*$ 修正、更新¶

$$ w_1 := w_1 - \alpha \frac{\mathrm \partial}{\mathrm \partial w_1} J(w_1) \\ $$

$w_1$ 修正的速度與 $\alpha$ 相關，$\alpha$ 稱為學習速率（Learning Rate）¶

推導對 $w_1$ 偏微分的成本函數（梯度）¶

$$ J(w_1) = \frac{1}{m}\sum_i (\hat{y} - y)_i^2 = \frac{1}{m}\sum_i (w_1x_{i1} - y_i)^2 \\ J(w_1) = \frac{1}{m}\sum_i (w_1^2x_{i1}^2-2w_1x_{i1}y_i+y_i^2) \\ \frac{\partial J}{\partial w_1} = \frac{1}{m}\sum_i(2w_1x_{i1}^2 - 2x_{i1}y_i) \\ \frac{\partial J}{\partial w_1} = \frac{2}{m}\sum_i(w_1x_{i1}^2 - x_{i1}y_i) = \frac{2}{m}\sum_i x_{i1}(x_{i1}w_1 - y_i) $$

梯度遞減（Gradient Descent）：從隨機指定的 $w_1$ 起始透過迭代、有意識地朝 $w_1^*$ 修正、更新¶

$$ w_1 := w_1 - \alpha \frac{\mathrm \partial}{\mathrm \partial w_1} J(w_1) \\ w_1 := w_1 - \alpha \frac{2}{m}\sum_i x_{i1}(x_{i1}w_1 - y_i) \\ w_1 := w_1 - \alpha \frac{2}{m}\sum_i x_{i1}(\hat{y_i} - y_i) $$

def get_grad(X, y, w_1):
    m = y.size
    y_hat = w_1*X
    err = y_hat - y
    grad = (X*(err)).sum()*2/m

透過先前的經驗，我們知道可以利用梯度遞減求 $w_0$ 與 $w_1$¶

$$ w_0 := w_0 - \alpha \frac{\mathrm \partial}{\mathrm \partial w_0} J(w_0, w_1) \\ w_1 := w_1 - \alpha \frac{\mathrm \partial}{\mathrm \partial w_1} J(w_0, w_1) $$

推導對 $w_0$ 與 $w_1$ 偏微分的成本函數（梯度）¶

$$ J(w_0, w_1) = \frac{1}{m}\sum_i(\hat{y_i} - y_i)^2 = \frac{1}{m}\sum_i(w_0 + w_1x_{i1} - y_i)^2 \\ J(w_0, w_1) = \frac{1}{m}\sum_i\big((w_0 + w_1x_{i1})^2-2(w_0 + w_1x_{i1})y_i+y_i^2\big) \\ J(w_0, w_1) = \frac{1}{m}\sum_i\big(w_0^2+2w_0w_1x_{i1}+w_1^2x_{i1}^2-2(w_0 + w_1x_{i1})y_i+y_i^2\big) \\ \frac{\partial J}{\partial w_0} = \frac{1}{m}\sum_i(2w_0+2w_1x_{i1}-2y_i) = \frac{2}{m}\sum_i(w_0+w_1x_{i1}-y_i) \\ \frac{\partial J}{\partial w_1} = \frac{1}{m}\sum_i(2w_0x_{i1}+2w_1x_{i1}^2-2x_{i1}y_i) = \frac{2}{m}\sum_i(w_0x_{i1}+w_1x_{i1}^2-x_{i1}y_i) = \frac{2}{m}\sum_i x_{i1}(w_0+w_1x_{i1}-y_i) \\ $$

梯度遞減（Gradient Descent）：從隨機指定的 $(w_0, w_1)$ 起始透過迭代、有意識地朝 $(w_0^*, w_1^*)$ 修正、更新¶

$$ w_0 := w_0 - \alpha \frac{\mathrm \partial}{\mathrm \partial w_0} J(w_0, w_1) = w_0 - \alpha \frac{2}{m}\sum_i (w_0+w_1x_{i1}-y_i) = w_0 - \alpha \frac{2}{m}\sum_i (\hat{y_i}-y_i)\\ w_1 := w_1 - \alpha \frac{\mathrm \partial}{\mathrm \partial w_1} J(w_0, w_1) = w_1 - \alpha \frac{2}{m}\sum_i x_{i1}(w_0+w_1x_{i1}-y_i) = w_1 - \alpha \frac{2}{m}\sum_i x_{i1}(\hat{y_i}-y_i)\\ $$

def get_grad(x_1, y, w_1, w_0):
    m = y.size
    y_hat = w_0 + w_1*x_1
    err = y_hat - y
    grad_0 = err.sum()*2/m
    grad_1 = ((x_1*err).sum())*2/m
    return grad_0, grad_1

最後我們將 $h$ 還原為泛化的外觀：¶

$$ h(x) = w_0x_0 + w_1x_1 + w_2x_2 + ... + w_nx_n = w^Tx \text{, where } x_0 = 1 \\ h(X) = Xw $$

透過先前的經驗，我們知道可以利用梯度遞減求 $w$¶

$$ w_0 := w_0 - \alpha \frac{\mathrm \partial}{\mathrm \partial w_0} J(w) \\ w_1 := w_1 - \alpha \frac{\mathrm \partial}{\mathrm \partial w_1} J(w) \\ .\\ .\\ .\\ w_n := w_n - \alpha \frac{\mathrm \partial}{\mathrm \partial w_1} J(w) $$

利用梯度遞減求 $w$，寫成泛化的外觀：¶

$$ w_i := w_i - \alpha \frac{\mathrm \partial}{\mathrm \partial w_i} J(w) \\ \text{where } i = 0, 1, 2, ..., n $$

推導對 $w_i$ 偏微分的成本函數（梯度）¶

$$ J(w) = \frac{1}{m}(\hat{y} - y)^T(\hat{y} - y) = \frac{1}{m}(Xw - y)^T(Xw-y) \\ J(w) = \frac{1}{m}(w^TX^T-y^T)(Xw-y) = \frac{1}{m}(w^TX^TXw-w^TX^Ty-y^TXw+y^Ty) = \frac{1}{m}(w^TX^TXw-2w^TX^Ty+y^Ty)\\ \frac{\partial J}{\partial w} = \frac{1}{m}(2X^TXw - 2X^Ty) = \frac{2}{m}(X^T(Xw - y))\\ $$

梯度遞減（Gradient Descent）：從隨機指定的 $(w)$ 起始透過迭代、有意識地朝 $(w^*)$ 修正、更新¶

$$ w := w - \alpha\frac{2}{m}(X^T(Xw - y)) \\ w := w - \alpha\frac{2}{m}(X^T(\hat{y} - y)) $$

$$w = (X^{(train)T}X^{(train)})^{-1}X^{(train)T}y^{(train)}$$

$X$ $m \times n$$X^T$ $n \times m$$X^TX$ $n \times n$

$$ A = \begin{bmatrix} a & b \\ c & d \\ \end{bmatrix} \\ A^{-1} = \frac{1}{ad-bc}\begin{bmatrix} d & -b \\ -c & a \\ \end{bmatrix} = \frac{1}{det(A)}\begin{bmatrix} d & -b \\ -c & a \\ \end{bmatrix}\\ $$

def gradient_descent(X, y, epochs, learning_rate):
    n = X.shape[1]
    w = np.random.rand(n, 1)
    j_history = []
    w_0_history, w_1_history, w_2_history = [w[0, 0]], [w[1, 0]], [w[2, 0]]
    n_prints = 20
    interval = epochs // n_prints
    for i in range(epochs):
        mse = get_mse(X, y, w)
        j_history.append(mse)
        grad = get_grad(X, y, w)
        w -= learning_rate*grad
        w_0_history.append(w[0, 0])

    return w, w_0_history, w_1_history, w_2_history, j_history

動量梯度遞減（Gradient Descent with Momentum）¶

$$ v_t := \gamma v_{t-1} + (1-\gamma) \nabla J(w) \\ w := w - \alpha v_t \\ \text{where, } \gamma = 0.9 $$

def momentum(X, y, epochs, learning_rate, gamma=0.9):
    n = X.shape[1]
    w = np.random.rand(n, 1)
    j_history = []
    w_0_history, w_1_history = [w[0, 0]], [w[1, 0]]
    n_prints = 20
    interval = epochs // n_prints
    # Initiate velocities
    velocity_values = np.array([0, 0], dtype=float).reshape(-1, 1)
    velocities = {i: velocity_values for i in range(epochs)}
    for i in range(epochs):
        mse = get_mse(X_train, y_train, w)
        j_history.append(mse)
        grad = get_grad(X_train, y_train, w)
        # Use velocities to update w
        if i == 0:
            velocities[i] = (1-gamma) * grad
        else:
            velocities[i] = gamma * velocities[i-1] + (1 - gamma) * grad
        w -= learning_rate*velocities[i]
        if i % interval == 0:
            print("{} iterations: w_0 = {}; w_1 = {}".format(i, w[0, 0], w[1, 0]))
            w_0_history.append(w[0, 0])
            w_1_history.append(w[1, 0])
    return w, w_0_history, w_1_history, j_history

Adam 梯度遞減（Gradient Descent with Adaptive Moment Estimation）¶

RMSProp

Source: https://wiki.komica.org/

結合動量梯度遞減與 RMSProp 梯度遞減的特性¶

$$ m_t := \beta_1 m_{t-1} + (1-\beta_1)\nabla J(w) \\ v_t := \beta_2 v_{t-1} + (1-\beta_2)\nabla J^2(w) \\ \hat{m_t} = \frac{m_t}{1-\beta_1^t} \\ \hat{v_t} = \frac{v_t}{1-\beta_2^t} \\ w := w - \alpha \frac{\hat{m_t}}{\epsilon + \sqrt{\hat{v_t}}} \\ \text{where, } \epsilon = 1e^{-8} \text{, }\beta_1 = 0.9 \text{, } \beta_2 = 0.999 $$

def adam(X, y, epochs, learning_rate, epsilon=1e-8, beta_1=0.9, beta_2=0.999):
    n = X.shape[1]
    w = np.random.rand(n, 1)
    j_history = []
    w_0_history, w_1_history = [w[0, 0]], [w[1, 0]]
    n_prints = 20
    interval = epochs // n_prints
    # Initiate first and second order moment generation function
    mt_values = np.array([0, 0], dtype=float).reshape(-1, 1)
    mts = {i: mt_values for i in range(epochs)}
    vt_values = np.array([0, 0], dtype=float).reshape(-1, 1)
    vts = {i: vt_values for i in range(epochs)}
    for i in range(epochs):
        mse = get_mse(X, y, w)
        j_history.append(mse)
        grad = get_grad(X, y, w)
        # Use adaptive momentums and velocities to update w
        if i == 0:
            mt = (1-beta_1)*grad
            vt = (1-beta_2)*grad**2
            mts[i] = mt
            vts[i] = vt
            mt_hat = mt / (1-beta_1**(i+1))
            vt_hat = vt / (1-beta_2**(i+1))
            ada_grad = mt_hat / (epsilon + vt_hat**0.5)
        else:
            mt = beta_1*mts[i-1] + (1.0-beta_1)*grad
            vt = beta_2*vts[i-1] + (1.0-beta_2)*grad**2
            mts[i] = mt
            vts[i] = vt
            mt_hat = mt / (1-beta_1**(i+1))
            vt_hat = vt / (1-beta_2**(i+1))
            ada_grad = mt_hat / (epsilon + vt_hat**0.5)
        w -= learning_rate*ada_grad
        if i % interval == 0:
            print("{} iterations: w_0 = {}; w_1 = {}".format(i, w[0, 0], w[1, 0]))
            w_0_history.append(w[0, 0])
            w_1_history.append(w[1, 0])
    return w, w_0_history, w_1_history, j_history
