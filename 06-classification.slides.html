import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LogisticRegression
from sklearn.metrics import accuracy_score

titanic = pd.read_csv("https://storage.googleapis.com/kaggle_datasets/Titanic-Machine-Learning-from-Disaster/train.csv")
X = titanic["Pclass"].values.reshape(-1, 1)
y = titanic["Survived"].values
X_train, X_validation, y_train, y_validation = train_test_split(X, y, test_size=0.33, random_state=42)
lr = LogisticRegression(solver='lbfgs')
lr.fit(X_train, y_train)
y_pred = lr.predict(X_validation)
acc = accuracy_score(y_validation, y_pred)
print("Accuracy: {:.2f}%".format(acc*100))

Accuracy: 69.83%

A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P if its performance at tasks in T, as measured by P, improves with experience E.

Tom Mitchel

T: Task¶

P: Performance¶

  • 跟 $f$ 長得很像的 $h$，分類與後來在球場上的位置相同
  • 跟 $f$ 長得不太像的 $h$，分類與後來在球場上的位置不相同

分類模型 in short¶

建立一個輸入向量 $x$ 能夠得到純量 $\hat{y}$ 輸入的系統 $h$，其中向量 $w$ 稱為是系統的參數、$\hat{y}$ 則是一個離散值，經由某些函數（$g$）將 $h$ 轉換而得

$$ \hat{y} = g(h(x)) = g(w^Tx) \\ \hat{y} = g(h(X)) = g(Xw) \\ $$

使用誤分類數評估 $h$ 跟 $f$ 的相似程度¶

$$J(h)_{test} = \sum^{n_{test}}\delta (\hat{y} \neq y_{test})$$

簡單貝氏分類器（Naive Bayes Classifier）¶

如何從連續型的輸出 $Xw$ 到離散型的輸出 $\hat{y}$¶

$$ \hat{y} = g(h(X)) = H(\sigma(Xw)) $$

其中 $\sigma$ 俗稱為激勵函數（Activation function），$H$ 稱為階躍函數（Step function）¶

  • 激勵函數將連續型輸出轉換為機率
  • 階躍函數將機率輸出轉換為離散值

分類器的輸出亦可以機率表示：給定 $x$ 其類別為 $C_i$ 的機率為何¶

$$ \sigma(x^Tw) = P(C_i | x) $$

以上都跟「貝氏」沒有關係¶

現在有一張已知是 Jack、Queen 或 King 的撲克牌覆蓋在桌面上，它是 King 的機率是多少？¶

$$ P(King|Face) = \frac{4}{12} $$

根據貝氏定理回答¶

$$ P(King|Face) = \frac{P(Face|King) \times P(King)}{P(Face)} = \frac{1 \times \frac{4}{52}}{\frac{12}{52}} $$

貝氏定理¶

$$ P(C_i|x) = \frac{P(x|C_i)P(C_i)}{P(x)} $$

其中 $P(C_i|x)$ 即我們俗稱的後驗機率（Posterior Probability）

Monty Hall Problem¶

Suppose you're on a game show, and you're given the choice of three doors: Behind one door is a car; behind the others, goats. You pick a door, say No. 1, and the host, who knows what's behind the doors, opens another door, say No. 3, which has a goat. He then says to you, "Do you want to pick door No. 2?" Is it to your advantage to switch your choice?

Source: https://en.wikipedia.org/wiki/Monty_Hall_problem

Source: https://en.wikipedia.org/wiki/Monty_Hall_problem

Source: https://www.youtube.com

事件的代號¶

$$ C_i: \text{車子在第 i 扇門} \\ H_i: \text{主持人開第 i 扇門} \\ X_i: \text{觀眾選第 i 扇門} $$

計算這兩個機率，然後決定是否要從 $X_1$ 更換至 $X_2$¶

$$ P(C_1|H_3, X_1) = ? \\ P(C_2|H_3, X_1) = ? $$
$$ P(C_1|H_3, X_1) = \frac{P(C_1, H_3, X_1)}{P(H_3, X_1)} = \frac{P(H_3|C_1, X_1)P(C_1, X_1)}{P(H_3, X_1)} = \frac{\frac{1}{2} \times P(C_1, X_1)}{P(H_3) \times P(X_1)} = \frac{\frac{1}{2} \times P(C_1) \times P(X_1)}{P(H_3) \times 1} = \frac{\frac{1}{2} \times \frac{1}{3}}{\frac{1}{2}} = \frac{1}{3} \\ P(C_2|H_3, X_1) = \frac{P(C_2, H_3, X_1)}{P(H_3, X_1)} = \frac{P(H_3|C_2, X_1)P(C_2, X_1)}{P(H_3, X_1)} = \frac{1 \times P(C_2, X_1)}{P(H_3) \times P(X_1)} = \frac{1 \times P(C_2) \times P(X_1)}{P(H_3) \times 1} = \frac{\frac{1}{3}}{\frac{1}{2}} = \frac{2}{3} $$

計算有 n 個特徵的後驗機率（泛化形式）¶

$$ P(C_i|x_0, x_1, ..., x_n) \propto P(x_0, x_1, ..., x_n|C_i)P(C_i) \propto P(C_i) \prod_{j=1}^n P(x_j|C_i) $$

計算訓練資料的後驗機率¶

$$ P(Survived|Pclass=1, Sex=female) = P(Survived) \times \frac{P(Pclass=1|Survived)}{P(Survived)} \times \frac{P(female|Survived)}{P(Survived)}\\ P(Survived|Pclass=1, Sex=male) = P(Survived) \times \frac{P(Pclass=1|Survived)}{P(Survived)} \times \frac{P(male|Survived)}{P(Survived)}\\ P(Survived|Pclass=2, Sex=female) = P(Survived) \times \frac{P(Pclass=2|Survived)}{P(Survived)} \times \frac{P(female|Survived)}{P(Survived)} $$

計算訓練資料的後驗機率（cont'd）¶

$$ P(Survived|Pclass=2, Sex=male) = P(Survived) \times \frac{P(Pclass=2|Survived)}{P(Survived)} \times \frac{P(male|Survived)}{P(Survived)}\\ P(Survived|Pclass=3, Sex=female) = P(Survived) \times \frac{P(Pclass=3|Survived)}{P(Survived)} \times \frac{P(female|Survived)}{P(Survived)}\\ P(Survived|Pclass=3, Sex=male) = P(Survived) \times \frac{P(Pclass=3|Survived)}{P(Survived)} \times \frac{P(male|Survived)}{P(Survived)}\\ $$

def get_y_pred(X, mu_dead, var_dead, mu_survived, var_survived, p_dead, p_survived):
    n_obs = X.shape[0]
    y_pred = np.zeros(n_obs, dtype=int)
    for i in range(n_obs):
        p_dead_age = get_posterior_p(X[i, :], mu_dead, var_dead, p_dead)
        p_survived_age = get_posterior_p(X[i, :], mu_survived, var_survived, p_survived)
        if p_dead_age >= p_survived_age:
            y_pred[i] = 0
        else:
            y_pred[i] = 1
    return y_pred

羅吉斯回歸分類器（Logistic Regression Classifier）¶

從連續型的輸出 $Xw$ 到離散型的輸出 $\hat{y}$¶

$$ \hat{y} = g(h(X)) = H(\sigma(Xw)) $$

其中 $\sigma$ 俗稱為激勵函數（Activation function），$H$ 稱為階躍函數（Step function）¶

  • 激勵函數將連續型輸出轉換為機率
  • 階躍函數將機率輸出轉換為離散值

激勵函數將連續型輸出轉換為機率，羅吉斯回歸使用 sigmoid 函數¶

$$\sigma(Xw) = \frac{1}{1+e^{-Xw}}$$$$\sigma(z) = \frac{1}{1+e^{-z}}$$

Sigmoid 函數的外觀¶

將機率轉換為 $\hat{y} \in {\{0, 1\}}$¶

$$ \hat{y} = H(\sigma(z), prob) = \begin{cases} 1 & \quad \text{if } \sigma(z)\geq prob\\ 0 & \quad \text{otherwise}\\ \end{cases} $$

拆解羅吉斯回歸分類器的步驟¶

  • 尋找 $h(X) = Xw$ 的 $w$
  • 將 $h(X)$ 的輸出 $z$ 作為 Sigmoid 函數的輸入，得到 $\sigma(z)$
  • 將 $\sigma(z)$ 作為 $H$ 的輸入，得到 $\hat{y}$

在迴歸問題中我們尋找 $h(X) = Xw$ 的 $w$ 是根據一個目標：讓成本函數 MSE 最小化¶

在分類問題中我們尋找 $h(X) = Xw$ 的 $w$ 則是依據：讓誤分類數最小化¶

$$J(h)_{test} = \sum^{n_{test}}\delta (\hat{y} \neq y_{test})$$

為了讓誤分類數最小化，得在尋找 $w$ 的過程中設計成本函數 $J$¶

$$ J(w) = \frac{1}{m}\big(-ylog\,(\sigma(Xw))-(1-y)log(1-\sigma(Xw))\big)\\ = \frac{1}{m}\big(-ylog(\sigma(z))-(1-y)log(1-\sigma(z))\big)\\ J(w) = \frac{1}{m}\big(-log(\sigma(z))\big) ,\text{ where y = 1} \\ J(w) = \frac{1}{m}\big(-log(1-\sigma(z))\big) ,\text{ where y = 0} $$

經過設計的成本函數 $J$ 稱為交叉熵（Cross-entropy）或 Log Loss¶

交叉熵（Cross-entropy）或 Log Loss 成本函數設計的巧妙處在於讓錯誤分類的成本趨近無限大¶

  • 真實的 $y$ 為 1，$\sigma(z)$ 若離 0 比較近則成本將趨近無限大
  • 真實的 $y$ 為 0，$\sigma(z)$ 若離 1 比較近則成本將趨近無限大

import numpy as np

def sigmoid(z):
    return(1 / (1 + np.exp(-z)))

def get_cross_entropy(X, y, w, epsilon=1e-8):
    m = y.size
    z = X.dot(w)
    h = sigmoid(z)
    y_reshaped = y.reshape(-1, 1)
    cost_y1 = -y_reshaped.T.dot(np.log(h+epsilon))
    cost_y0 = -((1-y_reshaped).T).dot(np.log(1-h+epsilon))
    cross_entropy = (cost_y1 + cost_y0) / m
    return cross_entropy[0, 0]

欲使用梯度遞減找到一組 $w$ 讓 $J(w)$ 最小化，得先計算梯度¶

$$ J(w) = -\frac{1}{m}\big(y\,log\,(\sigma(Xw))+(1-y)(\,log\,(1-\sigma(Xw))\big) \\ .\\ \text{...chain rules, the derivative of sigmoid function...} .\\ \frac{\mathrm \partial}{\mathrm \partial w} J(w) = \frac{1}{m}X^T\big(\sigma(Xw) - y\big) $$

import numpy as np

def get_grad(X, y, w):
    m = y.size
    y_reshaped = y.reshape(-1, 1)
    z = X.dot(w)
    h = sigmoid(z)
    grad = (1/m)*(X.T.dot(h-y_reshaped))
    return grad[0, 0]

自訂 adagrad() 函數¶

def adagrad(X, y, epochs, learning_rate, epsilon=1e-8):
    n = X.shape[1]
    w = np.random.rand(n, 1)
    j_history = []
    n_prints = 20
    interval = epochs // n_prints
    # Initiate sum of squared gradients
    ssg = np.zeros(n, dtype=float).reshape(-1, 1)
    for i in range(epochs):
        cross_entropy = get_cross_entropy(X, y, w)
        j_history.append(cross_entropy)
        grad = get_grad(X, y, w)
        # Use adaptive gradient to update w
        ssg += grad**2
        ada_grad = grad / (epsilon + ssg**0.5)
        w -= learning_rate*ada_grad
        if i % interval == 0:
            print("{} iterations: Cost: {}".format(i, cross_entropy))
    return w, j_history
