三個 Buzz-words¶

Source: rapidminer

機器學習是實踐人工智慧的手段，深度學習是機器學習中的一種方式¶

機器學習是透過程式讓電腦能夠從資料中學習的學科¶

說起來有點繞口令：Machine learning is the science of programming computers so they can learn from data.

精準地用一句話說明機器學習¶

A computer program is said to learn from experience E with respect to some task T and some performance measure P, if its performance on T, as measured by P, improves with experience E.

Tom Mitchell

經驗：Experience, E¶

  • 尋找已實現過的歷史資料，這些資料中具有輸入 $x$ 也具有輸出 $y$
  • 通常稱呼為具有標籤的資料（Labeled data）

任務：Task, T¶

  • 有些任務用固定的程式邏輯判斷、解決，例如判斷數字是否為質數
  • 有些任務很難用固定的程式邏輯判斷、解決，例如數值預測、類別預測
    • 有特徵可循：其他機器學習演算法
    • 無特徵可循：深度學習演算法

def is_prime(x):
    div_cnt = 0
    for i in range(1, x+1):
        if x % i == 0:
            div_cnt += 1
        if div_cnt > 2:
            break
    return int(div_cnt == 2)

勇者鬥惡龍：擊敗龍王的任務¶

Source: https://dragonquest.square-enix-games.com

機器學習：尋找 $h$ 的任務¶

於可及範圍的 $H$ 中挑選一個與 $f$ 最相似的 $h_i$

$$y = f(x)$$$$\boldsymbol{H} = \{h_1,h_2,\dotsc,h_n\}$$$$\hat{y_i} = h_i(x)$$

何謂可及範圍的 $H$？¶

  • 以類別預測為例，除了使用 Scikit-Learn 已經寫好的分類器 LogisticRegression（做為 $h_1$）
  • 亦可以自己定義一個分類器做為 $h_2$
  • 亦可以丟銅板決定做為 $h_3$
In [11]:
# h1
logit = LogisticRegression()
h1 = logit.fit(X_train, y_train)
y_hat_h1 = h1.predict(X_valid)

自己定義一個分類器做為 $h_2$¶

身高高於 2.03 公尺（6 呎 8 吋）的球員分類為前場，反之分類為後場。

In [12]:
# h2
class my_classifier:
    def predict(self, x):
        self.x = x
        return np.where(self.x >= 2.03, 1, 0).ravel()

h2 = my_classifier()
y_hat_h2 = h2.predict(X_valid)

該如何判斷可及範圍中的哪個 $h_i$ 與 $f$ 最相似呢？¶

  • 比對哪一個 $h_i$ 所預測的 $\hat{y_i}$ 與 $y$ 最相似
  • 進而推論哪個 $h_i$ 與 $f$ 最相似

評估：Performance, P¶

利用量化的指標來衡量哪個 $h_i$ 與 $f$ 最相似
