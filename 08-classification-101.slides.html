羅吉斯回歸分類器（Logistic Regression Classifier）¶

分類模型 in short¶

$x$ $\hat{y}$ $h$ $w$ $\hat{y}$ $g$ $h$

$$ \hat{y} = g(h(x)) = g(w^Tx) \\ \hat{y} = g(h(X)) = g(Xw) \\ $$

使用誤分類數評估 $h$ 跟 $f$ 的相似程度¶

$$J(h)_{test} = \sum^{n_{test}}\delta (\hat{y} \neq y_{test})$$

如何從連續型的輸出 $Xw$ 到離散型的輸出 $\hat{y}$¶

$$ \hat{y} = g(h(X)) = H(\sigma(Xw)) $$

其中 $\sigma$ 俗稱為激勵函數（Activation function），$H$ 稱為階躍函數（Step function）¶

分類器的輸出亦可以機率表示：給定 $x$ 其類別為 $C_i$ 的機率為何¶

$$ \sigma(x^Tw) = P(C_i | x) $$

激勵函數將連續型輸出轉換為機率，羅吉斯回歸使用 sigmoid 函數¶

$$\sigma(Xw) = \frac{1}{1+e^{-Xw}}$$$$\sigma(z) = \frac{1}{1+e^{-z}}$$

Sigmoid 函數的外觀¶

將機率轉換為 $\hat{y} \in {\{0, 1\}}$¶

$$ \hat{y} = H(\sigma(z), prob) = \begin{cases} 1 & \quad \text{if } \sigma(z)\geq prob\\ 0 & \quad \text{otherwise}\\ \end{cases} $$

拆解羅吉斯回歸分類器的步驟¶

  • $h(X) = Xw$ $w$
  • $h(X)$ $z$ Sigmoid $\sigma(z)$
  • $\sigma(z)$ $H$ $\hat{y}$

在迴歸問題中我們尋找 $h(X) = Xw$ 的 $w$ 是根據一個目標：讓成本函數 MSE 最小化¶

在分類問題中我們尋找 $h(X) = Xw$ 的 $w$ 則是依據：讓誤分類數最小化¶

$$J(h)_{test} = \sum^{n_{test}}\delta (\hat{y} \neq y_{test})$$

為了讓誤分類數最小化，得在尋找 $w$ 的過程中設計成本函數 $J$¶

$$ J(w) = \frac{1}{m}\big(-ylog\,(\sigma(Xw))-(1-y)log(1-\sigma(Xw))\big)\\ = \frac{1}{m}\big(-ylog(\sigma(z))-(1-y)log(1-\sigma(z))\big)\\ J(w) = \frac{1}{m}\big(-log(\sigma(z))\big) ,\text{ where y = 1} \\ J(w) = \frac{1}{m}\big(-log(1-\sigma(z))\big) ,\text{ where y = 0} $$

經過設計的成本函數 $J$ 稱為交叉熵（Cross-entropy）或 Log Loss¶

交叉熵（Cross-entropy）或 Log Loss 成本函數設計的巧妙處在於讓錯誤分類的成本趨近無限大¶

  • $y$ 1$\sigma(z)$ 0
  • $y$ 0$\sigma(z)$ 1

import numpy as np

def sigmoid(z):
    return(1 / (1 + np.exp(-z)))

def get_cross_entropy(X, y, w, epsilon=1e-8):
    m = y.size
    z = X.dot(w)
    h = sigmoid(z)
    y_reshaped = y.reshape(-1, 1)
    cost_y1 = -y_reshaped.T.dot(np.log(h+epsilon))
    cost_y0 = -((1-y_reshaped).T).dot(np.log(1-h+epsilon))
    cross_entropy = (cost_y1 + cost_y0) / m
    return cross_entropy[0, 0]

欲使用梯度遞減找到一組 $w$ 讓 $J(w)$ 最小化，得先計算梯度¶

$$ J(w) = -\frac{1}{m}\big(y\,log\,(\sigma(Xw))+(1-y)(\,log\,(1-\sigma(Xw))\big) \\ .\\ \text{...chain rules, the derivative of sigmoid function...} .\\ \frac{\mathrm \partial}{\mathrm \partial w} J(w) = \frac{1}{m}X^T\big(\sigma(Xw) - y\big) $$

import numpy as np

def get_grad(X, y, w):
    m = y.size
    y_reshaped = y.reshape(-1, 1)
    z = X.dot(w)
    h = sigmoid(z)
    grad = (1/m)*(X.T.dot(h-y_reshaped))
    return grad[0, 0]

def gradient_descent(X, y, epochs, learning_rate):
    n = X.shape[1]
    w = np.random.rand(n, 1)
    j_history = []
    n_prints = 20
    interval = epochs // n_prints
    for i in range(epochs):
        cross_entropy = get_cross_entropy(X, y, w)
        j_history.append(cross_entropy)
        grad = get_grad(X, y, w)
        w -= learning_rate*grad
        if i % interval == 0:
            print("{} iterations: cost = {}".format(i, cross_entropy))
    return w, j_history

def step_function(prob, prob_threshold=0.5):
    if prob >= prob_threshold:
        return 1
    else:
        return 0

def get_y_pred(X_validation, w):
    h = X_validation.dot(w)
    z = sigmoid(h)
    z_ravel = z.ravel()
    vfunc = np.vectorize(step_function)
    y_pred = vfunc(z_ravel)
    y_pred = y_pred.astype(int)
    return y_pred

分類模型的指標起源於混淆矩陣（Confusion Matrix）¶

Source: https://en.wikipedia.org/wiki/Confusion_matrix

混淆矩陣（Confusion Matrix）由四個象限組成¶

  • False positive $\hat{y_i} = 1$ $y_i = 0$
  • True positive $\hat{y_i} = 1$ $y_i = 1$
  • False negative $\hat{y_i} = 0$ $y_i = 1$
  • True negative $\hat{y_i} = 0$ $y_i = 0$

準確率（Accuracy）¶

$$ accuracy = \frac{\text{TP + TN}}{\text{TP + TN + FP + FN}} $$

Precision¶

Precision

$$precision = \frac{\text{TP}}{\text{TP + FP}}$$

Recall¶

Recall

$$ recall = \frac{\text{TP}}{\text{TP + FN}} $$

多數的時候我們使用 Accuracy 評估分類模型的表現，但是 Accuracy 在陽性樣本比例極低的時候會失準，這時 Precision 與 Recall 就派上用場¶

  • Precision FP
  • Recall FN

$F_1$ score¶

Precision Recall

$$ F_1 = 2 \times \frac{precision \times recall}{precision + recall} $$

泛化的 $F$ score¶

$\beta$ Precision Recall $\beta = 1$ $F_1$$\beta = 0$ Precision$\beta = \infty$ Recall

$$ F = (1+\beta^2) \times \frac{precision \times recall}{(\beta^2 \cdot precision) + recall} $$

AUC（Area under the ROC Curve）¶

$Xw$ $\hat{y}$

$$ \hat{y} = g(h(X)) = H(\sigma(Xw)) \\ $$

階躍函數必須給定一個 Threshold（門檻值）¶

$$ \hat{y} = H(\sigma(z), threshold) = \begin{cases} 1 & \quad \text{if } \sigma(z)\geq threshold\\ 0 & \quad \text{otherwise}\\ \end{cases} $$

給定一個二元分類模型和不同門檻值，就能從所有樣本的真實值和預測值繪製出一個橫軸為 FPR、 縱軸為 Precision 的 ROC 空間¶

$$ FPR = \frac{FP}{FP + TN} \\ TPR = \frac{TP}{TP + FN} $$

ROC Curve 底下的面積稱為 AUC，最理想的情況是 AUC 為 1；而 AUC 為 0.5，則是代表隨機挑選、沒有預測能力的分類模型，大部分情形都是落在這兩種之間¶
