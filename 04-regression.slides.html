P: Performance¶

  • $f$ $h$
  • $f$ $h$

線性迴歸模型 in short¶

$x$ $\hat{y}$ $h$$x$ $\hat{y}$ $w$

$$\hat{y} = h(x) = w_0 + w_1 x_1 + w_2 x_2 + ... + w_n x_n \\ \hat{y} = h(x) = w_0 x_0 + w_1 x_1 + w_2 x_2 + ... + w_n x_n \text{ , where } x_0 = 1 \\ \hat{y} = h(x) = w^Tx$$

將 $h$ 表示得更廣泛¶

$$\hat{y} = h(X) = \begin{bmatrix} x_{00}, x_{01}, ..., x_{0n} \\ x_{10}, x_{11}, ..., x_{1n} \\.\\.\\.\\ x_{m0}, x_{m1}, ..., x_{mn} \end{bmatrix} \begin{bmatrix} w_0 \\ w_1 \\.\\.\\.\\ w_n \end{bmatrix} = Xw$$

$w$：系統的參數¶

  • $w_i$ $x_i$ $\hat{y}$
  • $w_i$ $x_i$ $\hat{y}$
  • $w_i$ $x_i$ $\hat{y}$

使用 MSE（Mean Squared Error）評估 $h$ 跟 $f$ 的相似程度¶

$$MSE_{test} = \frac{1}{m}\sum_i(\hat{y}^{(test)} - y^{(test)})_i^2$$

球季結束，將所有已經完成簽約自由球員（Free agents）跟新人（Rookies）的場均得分輸入 $h$，獲得年薪 $\hat{y}^{(test)}$，跟實際簽約金額 $y^{(test)}$ 比較¶

當預測年薪 $\hat{y}^{(test)}$，跟實際簽約金額 $y^{(test)}$ 相同，$MSE_{test}$ 為零¶

看公式之前複習幾個向量、矩陣相乘的特性¶

  • $x$ $y$ $x^Ty$ $y^Tx$
  • $A$ $B$ $(AB)^T = B^TA^T$
  • $Ax = b$ $x = A^{-1}b$

$x$ 與 $y$ 兩個維度相同的向量相乘為 $x^Ty$、亦等於 $y^Tx$¶

機器學習的任務：找出一組向量 $w$ 讓訓練資料的 $MSE_{train}$ 最小¶

$$\bigtriangledown_w MSE_{train} = 0 \\ \bigtriangledown_w \frac{1}{m} (y^{(train)} - y^{(train)})^2 = 0 \\ \frac{1}{m} \bigtriangledown_w (X^{(train)}w - y^{(train)})^2 = 0 \\ \bigtriangledown_w (X^{(train)}w - y^{(train)})^T(X^{(train)}w - y^{(train)}) = 0 \\ \frac{\partial}{\partial w} (w^TX^{(train)T}X^{(train)}w - 2w^TX^{(train)T}y^{(train)} + y^{(train)T}y^{(train)}) = 0 \\ 2X^{(train)T}X^{(train)}w - 2X^{(train)T}y^{(train)} = 0 \\ w = (X^{(train)T}X^{(train)})^{-1}X^{(train)T}y^{(train)} $$

這個求解稱為 normal equations，又稱 closed form solution¶

$$w = (X^{(train)T}X^{(train)})^{-1}X^{(train)T}y^{(train)}$$

import numpy as np

def get_h_plot(w):
    X_pred = np.linspace(X_train.min(), X_train.max(), 100)
    X_0 = np.ones(X_pred.size).reshape(-1, 1)
    X_pred_X_0 = np.concatenate([X_0, X_pred.reshape(-1, 1)], axis=1)
    y_pred = X_pred_X_0.dot(w)
    plt.scatter(X_train, y_train, label="train", s=5)
    plt.plot(X_pred, y_pred, label="h", c="red", linewidth=2)
    plt.legend()

看起來後衛（G）與前鋒（F）的 $h$ 相似，但中鋒（C）比較自成一格¶

我們能否依據 pos_recategorized 變數將 $h$ 調整為¶

$$\text{if pos_recategorized=='G':}\quad h^{(G)}(x) = w_0^{(G)} + w_1^{(G)} x_1\\ \text{if pos_recategorized=='F':}\quad h^{(F)}(x) = w_0^{(F)} + w_1^{(F)} x_1\\ \text{if pos_recategorized=='C':}\quad h^{(C)}(x) = w_0^{(C)} + w_1^{(C)} x_1\\ $$

將三個 if 條件所對應的 $h(x)$ 利用一個 $\delta$ 係數結合¶

$$h(x) = w_0^{(G)}\delta^{(G)} + w_1^{(G)}\delta^{(G)} x_1 + \\ w_0^{(F)}\delta^{(F)} + w_1^{(F)}\delta^{(F)} x_1 + \\ w_0^{(C)}\delta^{(C)} + w_1^{(C)}\delta^{(C)} x_1 $$

$\delta$ 係數在不同鋒衛位置時¶

$$\delta^{(G)} = 1 \quad \text{if pos_recategorized=='G', otherwise} \quad \delta^{(G)} = 0\\ \delta^{(F)} = 1 \quad \text{if pos_recategorized=='F', otherwise} \quad \delta^{(F)} = 0 \\ \delta^{(C)} = 1 \quad \text{if pos_recategorized=='C', otherwise} \quad \delta^{(C)} = 0 $$

pos_recategorized=='G'¶

$$h(x) = w_0^{(G)} + w_1^{(G)}x_1 $$

pos_recategorized=='F'¶

$$h(x) = w_0^{(F)} + w_1^{(F)}x_1 $$

pos_recategorized=='C'¶

$$h(x) = w_0^{(C)} + w_1^{(C)}x_1 $$

正規化（Regularization）¶

$h$

$$MSE_{train} = \frac{1}{m}\sum_i(w^Tx^{train} - y^{(train)})_i^2 + \lambda \sum_{i} w_i^2 \\ MSE_{train} = \frac{1}{m}(X^{(train)}w - y^{(train)})^T(X^{(train)}w - y^{(train)}) + \lambda w^Tw $$

具備正規化的 Normal equations¶

$$ \bigtriangledown_w MSE_{train} = 0 \\ \bigtriangledown_w \frac{1}{m}(X^{(train)}w - y^{(train)})^T(X^{(train)}w - y^{(train)}) + \lambda w^Tw = 0 \\ \frac{\partial}{\partial w}\frac{1}{m} (w^TX^{(train)T}X^{(train)}w - 2w^TX^{(train)T}y^{(train)} + y^{(train)T}y^{(train)}) + \lambda w^Tw = 0 \\ \frac{1}{m} (2X^{(train)T}X^{(train)}w - 2X^{(train)T}y^{(train)}) + 2\lambda w = 0 \\ \frac{1}{m} X^{(train)T}X^{(train)}w + \lambda w = \frac{1}{m} X^{(train)T}y^{(train)} \\ (\frac{1}{m}X^{(train)T}X^{(train)} + \lambda I)w = \frac{1}{m} X^{(train)T}y^{(train)} \\ w = (\frac{1}{m}X^{(train)T}X^{(train)} + \lambda I)^{-1}\frac{1}{m} X^{(train)T}y^{(train)} $$
