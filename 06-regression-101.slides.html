線性迴歸模型 in short¶

$x$ $\hat{y}$ $h$$x$ $\hat{y}$ $w$

$$\hat{y} = h(x) = w_0 + w_1 x_1 + w_2 x_2 + ... + w_n x_n \\ \hat{y} = h(x) = w_0 x_0 + w_1 x_1 + w_2 x_2 + ... + w_n x_n \text{ , where } x_0 = 1 \\ \hat{y} = h(x) = w^Tx$$

將 $h$ 表示得更廣泛¶

$$\hat{y} = h(X) = \begin{bmatrix} x_{00}, x_{01}, ..., x_{0n} \\ x_{10}, x_{11}, ..., x_{1n} \\.\\.\\.\\ x_{m0}, x_{m1}, ..., x_{mn} \end{bmatrix} \begin{bmatrix} w_0 \\ w_1 \\.\\.\\.\\ w_n \end{bmatrix} = Xw$$

$w$：系統的參數¶

  • $w_i$ $x_i$ $\hat{y}$
  • $w_i$ $x_i$ $\hat{y}$
  • $w_i$ $x_i$ $\hat{y}$

使用 MSE（Mean Squared Error）評估 $h$ 跟 $f$ 的相似程度¶

$$MSE_{test} = \frac{1}{m}\sum_i(\hat{y}^{(test)} - y^{(test)})_i^2$$

將所有球員身高輸入 $h$，獲得預測體重 $\hat{y}^{(test)}$，跟實際體重 $y^{(test)}$ 比較¶

當預測體重 $\hat{y}^{(test)}$，跟體重 $y^{(test)}$ 相同，$MSE_{test}$ 為零¶

機器學習的任務：找出一組向量 $w$ 讓訓練資料的 $MSE_{train}$ 最小¶

$$\bigtriangledown_w MSE_{train} = 0 \\ \bigtriangledown_w \frac{1}{m}\sum_i(\hat{y}^{(train)} - y^{(train)})_i^2 = 0\\ \frac{1}{m} \bigtriangledown_w (\hat{y}^{(train)} - y^{(train)})^T(\hat{y}^{(train)} - y^{(train)}) = 0\\ \bigtriangledown_w (X^{(train)}w - y^{(train)})^T(X^{(train)}w - y^{(train)}) = 0 \\ \frac{\partial}{\partial w} (w^TX^{(train)T}X^{(train)}w - 2w^TX^{(train)T}y^{(train)} + y^{(train)T}y^{(train)}) = 0 \\ 2X^{(train)T}X^{(train)}w - 2X^{(train)T}y^{(train)} = 0 \\ w = (X^{(train)T}X^{(train)})^{-1}X^{(train)T}y^{(train)} $$

這個求解稱為 normal equations，又稱 closed form solution¶

$$w = (X^{(train)T}X^{(train)})^{-1}X^{(train)T}y^{(train)}$$

給定 $X_{train}$ 與 $y_{train}$ 利用 normal equations 求 $w$¶

In [11]:
def get_w_ne(X_train, y_train):
    X_train_T = np.transpose(X_train)
    X_train_T_X_train = X_train_T.dot(X_train)
    X_train_T_X_train_inverse = np.linalg.inv(X_train_T_X_train)
    w = X_train_T_X_train_inverse.dot(X_train_T.dot(y_train))
    return w

看起來後衛（G）與前鋒（F）的 $h$ 相似，但中鋒（C）比較自成一格¶

我們能否依據 pos_recategorized 變數將 $h$ 調整為¶

$$\text{if pos_recategorized=='G':}\quad h^{(G)}(x) = w_0^{(G)} + w_1^{(G)} x_1\\ \text{if pos_recategorized=='F':}\quad h^{(F)}(x) = w_0^{(F)} + w_1^{(F)} x_1\\ \text{if pos_recategorized=='C':}\quad h^{(C)}(x) = w_0^{(C)} + w_1^{(C)} x_1\\ $$

將三個 if 條件所對應的 $h(x)$ 利用一個 $\delta$ 係數結合¶

$$h(x) = w_0^{(G)}\delta^{(G)} + w_1^{(G)}\delta^{(G)} x_1 + \\ w_0^{(F)}\delta^{(F)} + w_1^{(F)}\delta^{(F)} x_1 + \\ w_0^{(C)}\delta^{(C)} + w_1^{(C)}\delta^{(C)} x_1 $$

$\delta$ 係數在不同鋒衛位置時¶

$$\delta^{(G)} = 1 \quad \text{if pos_recategorized=='G', otherwise} \quad \delta^{(G)} = 0\\ \delta^{(F)} = 1 \quad \text{if pos_recategorized=='F', otherwise} \quad \delta^{(F)} = 0 \\ \delta^{(C)} = 1 \quad \text{if pos_recategorized=='C', otherwise} \quad \delta^{(C)} = 0 $$

pos_recategorized=='G'¶

$$h(x) = w_0^{(G)} + w_1^{(G)}x_1 $$

pos_recategorized=='F'¶

$$h(x) = w_0^{(F)} + w_1^{(F)}x_1 $$

pos_recategorized=='C'¶

$$h(x) = w_0^{(C)} + w_1^{(C)}x_1 $$

正規化（Regularization）¶

$h$

$$MSE_{train} = \frac{1}{m}\sum_i(w^Tx^{train} - y^{(train)})_i^2 + \lambda \sum_{i} w_i^2 \\ MSE_{train} = (X^{(train)}w - y^{(train)})^T(X^{(train)}w - y^{(train)}) + \lambda w^Tw $$

具備正規化的 Normal equations¶

$$ \bigtriangledown_w MSE_{train} = 0 \\ \bigtriangledown_w \big((X^{(train)}w - y^{(train)})^T(X^{(train)}w - y^{(train)}) + \lambda w^Tw\big) = 0 \\ \frac{\partial}{\partial w} \big(w^TX^{(train)T}X^{(train)}w - 2w^TX^{(train)T}y^{(train)} + y^{(train)T}y^{(train)} + \lambda w^Tw\big) = 0 \\ 2X^{(train)T}X^{(train)}w - 2X^{(train)T}y^{(train)} + 2\lambda w = 0 \\ X^{(train)T}X^{(train)}w + \lambda w = X^{(train)T}y^{(train)} \\ (X^{(train)T}X^{(train)} + \lambda I)w = X^{(train)T}y^{(train)} \\ w = (X^{(train)T}X^{(train)} + \lambda I)^{-1} X^{(train)T}y^{(train)} $$

import numpy as np

def get_w_ne_reg(X_train, y_train, lambda_coef):
    n = X_train.shape[1]
    I = np.eye(n)
    left_mat = X_train.T.dot(X_train) + lambda_coef*I
    right_mat = X_train.T.dot(y_train)
    w = np.linalg.inv(left_mat).dot(right_mat)
    return w
